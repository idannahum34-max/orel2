# OREL Candles — Static Vercel Deploy

זו גרסה סטטית ונקייה לחלוטין.

אין npm.
אין package.json.
אין build.
אין dependencies.
אין install.

## העלאה ל-GitHub
מעלים את התיקייה `orel-candles-static-vercel` ל-GitHub.

## חיבור ל-Vercel
ב-Vercel בוחרים את ה-Repository.

הגדרות:
- Framework Preset: Other
- Build Command: להשאיר ריק
- Output Directory: להשאיר ריק
- Install Command: להשאיר ריק

Vercel יגיש ישירות את `index.html`.

## עדכון פרטים
כל האתר נמצא בקובץ `index.html`.
התמונות נמצאות בתיקיית `imgs`.
